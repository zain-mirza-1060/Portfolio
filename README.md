# Zain Mirza | Full Stack AI Engineer Portfolio

A modern, responsive portfolio website built with Next.js and Tailwind CSS, showcasing experience, projects, and skills.

## Features

- Responsive design for all devices
- Black and blue theme with smooth animations
- Content-driven from editable markdown files
- Experience, Industry Projects, Personal Projects sections
- Tech stack and contact info
- Headshot placeholder (initials)

## Content

Edit the markdown files in `content/` to update the site. The resume PDF in `content/Zain_resume.pdf` is the source of truth.

- `content/intro.md` - Name, title, tagline, summary
- `content/experience.md` - Work experience
- `content/industry-projects.md` - Industry projects
- `content/personal-projects.md` - Personal projects
- `content/tech-stacks.md` - Technologies
- `content/contact.md` - Email, phone, LinkedIn, GitHub

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- Gray Matter (markdown parsing)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
